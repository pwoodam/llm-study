# Simple Calculator

## 프로젝트 소개

이 프로젝트는 Python의 기본 프로젝트 구조와 모듈 분리 방식을 학습하기 위해 만든 간단한 계산기 애플리케이션입니다.

## 학습 목표

- Python 프로젝트 구조를 이해한다.
- 파일을 분리하고 모듈로 관리하는 방법을 익힌다.
- Git을 이용한 프로젝트 관리 흐름을 연습한다.

## 프로젝트 구조

```text
simple-calculator/
├── app/
│   ├── calculator.py
│   └── main.py
├── tests/
├── README.md
└── requirements.txt
```

## 제공 기능

- 덧셈 기능
- 뺄셈 기능

## 실행 환경

- Python 3.11.15
- Conda
- VS Code

## 실행 방법

다음 명령으로 프로그램을 실행할 수 있습니다.

```bash
python app/main.py
```

실행 예시:

```text
30
15
```
